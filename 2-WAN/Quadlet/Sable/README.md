## Dependencies

These are the services that this service depends on in my deployment.<br>
They're not noted as explicit requirements with `Requires=` but are needed for it to function fully.

- `Tuwunel` type: Matrix Homeserver
- `Sygnal` type: Notifications
- `PocketID` type: Auth
